# Portfolio Website

This is a ready-to-use portfolio website for **Muhammed Shereef K**.

## Files
- `index.html` – main portfolio page
- `style.css` – styling
- `script.js` – certificate filters and rendering
- `certificates/` – copied PDF certificates from your uploaded ZIP

## How to use
1. Open `index.html` in your browser.
2. Edit contact links in the **Contact** section.
3. Update project descriptions, skills, or about text if needed.
4. Upload the whole folder to GitHub Pages, Netlify, or Vercel.

## Easy hosting
### GitHub Pages
- Create a new GitHub repository
- Upload all files
- Go to **Settings > Pages**
- Deploy from the main branch

### Netlify / Vercel
- Drag and drop the folder or connect your GitHub repository
